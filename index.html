What's inside

Universal task orchestrator with 15+ integrated tools for automation, development, crypto operations, and system monitoring.

⏱
Task Scheduler
Cron-based scheduler for Python, Node.js, C# CSX, PowerShell, Bash. Parallel execution, retry logic, live output streaming via SSE.
🔧
C# Scripting (CSX)
Roslyn in-process executor with #load directives, SHA256 cache, IBrowserInstance abstraction. Compatible with ZennoPoster scripts.
🌐
HTTP Traffic Analyzer
Capture and replay requests. Generate code (C#, Python, TypeScript, cURL). Auto-detect security tokens. API skeleton generator.
💎
Web3 Treasury
Multi-chain wallet tracking, DeBank DeFi integration, Rabby automation, HD wallet generation (BIP39/BIP44), Nethereum EVM ops.
🎭
Browser Automation
Playwright 1.58.0 integration. WebSocket endpoint, profile persistence, traffic interception. Optional ZennoBrowser support.
🤖
AI Reports
OpenAI-compatible API integration (aiio, omniroute). Project analysis, system audits, code documentation generation.
📊
System Monitor
Native C# system snapshot: TCP tables, memory, processes. Sortable UI tables with AI audit endpoint.
📝
JSON/XML/Text Tools
JSON viewer with token detection, XML parser, Base64/URL/C# encoding tools. Code snippet library (Cliplates).
📡
Live Logs
Centralized log streaming with HTTP sink. Real-time updates via SSE. Filterable, searchable, exportable.
🔗
Optional Integrations
ZennoPoster V7, ZennoBrowser, Gmail API, Git operations, Captcha services. Modular architecture.
