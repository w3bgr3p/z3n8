
Конвертеры строк для подготовки данных при разработке ZP-шаблонов и скриптов.

Путь: `/text.html`

---

## URL Encode / Decode

Конвертирует строку в URL-encoded формат и обратно (`encodeURIComponent` / `decodeURIComponent`).

**Использование:**

- Вставь строку в левое поле → правое заполнится автоматически.
- Вставь encoded строку в правое поле → левое заполнится автоматически.
- Кнопки **→** и **←** — конвертировать принудительно.
- **Copy encoded** — скопировать результат.
- **✕ Clear** — очистить оба поля.

**Пример:** `Hello World / foo=bar` → `Hello%20World%20%2F%20foo%3Dbar`

## C# String Escaper

Превращает произвольную строку в C# string literal с экранированием спецсимволов.

Экранирует: `\`, `"`, `\n`, `\r`, `\t`, `\0`.

**Режим verbatim** (`@"..."`) — чекбокс. В этом режиме только удваиваются кавычки `"` → `""`. Удобно для путей к файлам.

**Использование:**

- Вставь строку в левое поле → правое покажет готовый C# литерал.
- Вставь C# литерал в правое → левое восстановит оригинал.
- **Copy** — скопировать результат.
- **✕ Clear** — очистить.

**Примеры:**

|Исходная строка|Результат (обычный)|Результат (verbatim)|
|---|---|---|
|`C:\Users\test`|`"C:\\Users\\test"`|`@"C:\Users\test"`|
|`He said "hi"`|`"He said \"hi\""`|`@"He said ""hi"""`|

## Base64 Encode / Decode

Конвертирует текст в Base64 и обратно (UTF-8).

**Использование:**

- Левое поле → правое: encode.
- Правое поле → левое: decode.
- Кнопки **→** и **←** — конвертировать принудительно.
- **Copy encoded** — скопировать результат.
- **✕ Clear** — очистить.

Ошибка `Invalid Base64` появляется если в правом поле невалидная строка.

---

## JSON String Escape

Превращает строку в JSON-safe значение для вставки внутрь JSON-документа.

**Использование:**

- Левое поле → правое: escape (через `JSON.stringify`).
- Правое поле → левое: unescape (через `JSON.parse`).
- **Copy** — скопировать результат.
- **✕ Clear** — очистить.

**Пример:** `He said "hello"` → `"He said \"hello\""`
