
Встроенная справка. Отображает документацию из Obsidian vault в виде интерактивного графа связей.

**Путь:** `/docs`  
**Хоткей:** `Alt+H`

---

## Обзор

Docs — встроенный просмотрщик документации, который рендерит содержимое Obsidian vault (по умолчанию `docs-vault`) в виде интерактивного графа связей между документами.

**Особенность:** Можно просматривать любой Obsidian vault, указав путь к нему в конфигурации.

---

## Функционал

### Граф связей

Визуализация связей между документами vault через Obsidian-ссылки `[[Document Name]]`.

- **Узлы** — документы (`.md` файлы)
- **Рёбра** — ссылки между документами
- **Интерактивность:**
  - Zoom — колесо мыши
  - Pan — перетаскивание
  - Клик на узел — открыть документ
  - Hover — показать превью

### Просмотр документов

При клике на узел графа открывается содержимое документа:

- Рендеринг Markdown
- Поддержка Obsidian-ссылок
- Навигация по ссылкам внутри документа
- Breadcrumbs для возврата назад

### Поиск

Поиск по названиям документов и содержимому:

- Фильтрация узлов графа
- Подсветка найденных документов
- Быстрый переход к результатам

---

## Конфигурация

По умолчанию используется vault `docs-vault` в корне проекта z3nIO.

Для просмотра другого vault:

1. Открой [[Config]]
2. Укажи путь к vault в настройках (если такая опция доступна)
3. Или измени путь в конфигурационном файле сервера

---

## Структура docs-vault

Текущий vault содержит:

- **index.md** — главная страница с обзором z3nIO
- **Navigation/Pages/** — документация страниц
- **Onboarding/** — руководства по первому запуску
  - 01. Настройка сервера
  - 02. Добавление машины воркера
  - 03. SAFU — Архитектура

## Edge cases

**Граф не отображается** — проверь что vault содержит `.md` файлы с Obsidian-ссылками `[[...]]`.

**Документ не открывается** — убедись что файл существует и имеет расширение `.md`.

**Ссылки не работают** — проверь синтаксис Obsidian-ссылок: `[[Document Name]]` или `[[Document Name#Section]]`.

**Пустая страница** — возможно путь к vault неверный или vault пустой. Проверь логи сервера.

**Граф показывает не тот vault** — по умолчанию используется `docs-vault` из корня проекта. Для просмотра другого vault измени конфигурацию.
