

Визуализация графа зависимостей C# репозитория. Генерирует интерактивный граф классов, методов и их связей.

**Путь:** `/graph.html`  
**Хоткей:** `Alt+G`

---

## Обзор

cs-graph — инструмент для анализа структуры C# проектов. Парсит исходный код, строит граф зависимостей и отображает его в интерактивном виде через iframe.

---

## Интерфейс

### Header

**cs-graph** — название страницы.

**Open in tab** — открыть граф в отдельной вкладке браузера (доступна после генерации).

### Toolbar

|Элемент|Описание|
|---|---|
|**repo path**|Путь к корню C# репозитория (например `C:\path\to\repo`)|
|**Generate**|Запустить генерацию графа|
|**Reload**|Перезагрузить iframe с графом (доступна после генерации)|

### Status Bar

Показывает текущее состояние:

|Индикатор|Описание|
|---|---|
|Серая точка|Нет данных|
|Желтая точка (мигает)|Генерация в процессе|
|Зеленая точка|Граф успешно сгенерирован|
|Красная точка|Ошибка генерации|

Текст справа от точки — детали статуса или путь к репозиторию.

## Использование

### Генерация графа

1. Введи путь к корню C# репозитория в поле **repo path**
2. Нажми **Generate**
3. Дождись завершения (статус изменится на зеленый)
4. Граф отобразится в iframe

**Enter** в поле repo path — запускает генерацию.

### Навигация по графу

После генерации iframe загружает интерактивный граф:

- **Zoom** — колесо мыши
- **Pan** — перетаскивание мышью
- **Клик на узел** — выделить узел и связанные элементы
- **Hover** — показать детали узла

### Открытие в отдельной вкладке

Кнопка **Open in tab** открывает граф в новой вкладке браузера. Полезно для работы с большими графами на втором мониторе.

## Backend

### Генерация

POST-запрос на `/graph/generate?repoPath={path}`:

1. Сервер парсит C# файлы в указанной директории
2. Строит граф зависимостей (классы, методы, вызовы)
3. Генерирует HTML с визуализацией (вероятно через D3.js или аналог)
4. Сохраняет результат

Ответ:

```json
{
  "repoPath": "C:\\path\\to\\repo"
}
```

Или при ошибке:

```json
{
  "error": "Error message"
}
```

### Отображение

GET-запрос на `/graph` возвращает сгенерированный HTML с графом.

Iframe загружает `/graph?{timestamp}` для обхода кэша браузера.

## Edge cases

**"repoPath is required"** — поле repo path пустое. Введи путь к репозиторию.

**Ошибка генерации** — проверь что:
- Путь существует и доступен
- Директория содержит C# файлы (`.cs`)
- У процесса z3nIO есть права на чтение директории

**Граф не отображается** — проверь консоль браузера (F12). Возможно ошибка парсинга или рендеринга.

**Граф пустой** — репозиторий не содержит C# файлов или парсер не смог их обработать.

**Reload не работает** — кнопка доступна только после успешной генерации. Если граф не загрузился — попробуй **Generate** заново.

**Open in tab открывает пустую страницу** — граф не был сгенерирован. Сначала нажми **Generate**.
