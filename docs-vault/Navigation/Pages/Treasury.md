

Управление криптовалютными кошельками. Мониторинг балансов по множеству блокчейнов, автоматическое обновление данных, swap и bridge операции.

**Путь:** `/treasury.html`  
**Хоткей:** `Alt+8`

---

## Обзор

Treasury — централизованная панель для отслеживания криптовалютных активов по множеству аккаунтов и блокчейнов. Отображает балансы в виде GitHub-style heatmap, агрегирует статистику по токенам и сетям, поддерживает AI-анализ портфеля.

---

## Header Stats

Верхняя панель показывает общую статистику:

|Поле|Описание|
|---|---|
|Accounts|Общее количество аккаунтов|
|Active|Аккаунты с ненулевым балансом|
|Chains|Количество активных блокчейнов|
|Total|Общая стоимость портфеля в USD|

**Updating...** — индикатор активного обновления балансов.

## Toolbar

|Элемент|Описание|
|---|---|
|**Refresh**|Перезагрузить данные из БД|

|**Max ID**|Максимальный ID аккаунта для отображения (1-10000)|

## Heatmap Table

Основная таблица с аккаунтами и балансами.

### Структура

- **ID** — номер аккаунта (sticky column)
- **Address** — адрес кошелька (sticky column, кликабельная ссылка на DeBank)
- **Колонки блокчейнов** — каждая сеть отображается как колонка с иконкой
- **TTL** — общая сумма по всем сетям для аккаунта
- **SWP** — кнопка для swap всех токенов в нативный токен сети
- **BRG** — кнопка для bridge токенов в выбранную сеть

### Heatmap Cells

Ячейки окрашены по сумме в USD:

|Цвет|Диапазон|
|---|---|
|Пустая (прозрачная)|$0|
|`level-1` (темно-зеленый)|$0.01 - $0.99|
|`level-2` (зеленый)|$1 - $9.99|
|`level-3` (светло-зеленый)|$10 - $99.99|
|`level-4` (яркий зеленый)|$100+|

**Hover** — увеличивает ячейку и показывает tooltip с детализацией токенов.

### Tooltip

При наведении на ячейку показывается:

- Номер аккаунта и название сети
- Адрес кошелька
- Список токенов с символом, количеством и стоимостью в USD
- Общая сумма по сети

## Sidebar

Правая панель с аналитикой и AI-отчетом.

### Top Tokens

Топ-15 токенов по стоимости:

- Символ токена
- Количество аккаунтов, владеющих токеном
- Общая стоимость в USD
- Процент от общего портфеля
- Progress bar

### Chains Distribution

Топ-10 блокчейнов по стоимости активов:

- Иконка сети
- Название сети
- Общая стоимость в USD

### Portfolio Summary

Общая статистика:

|Поле|Описание|
|---|---|
|Total Value|Общая стоимость портфеля|
|Unique Tokens|Количество уникальных токенов|
|Unique Chains|Количество уникальных блокчейнов|
|Active Accounts|Аккаунты с балансом > 0|

### AI Report

AI-анализ портфеля через LLM.

**Model** — выбор модели из списка доступных (загружается из `/ai-report/api/models`).

**Analyze** — запустить анализ. Отправляет данные портфеля в `/treasury/ai-analyze`.

**Clear** — очистить кэшированный результат.

Результат показывает:

- Текст анализа
- Использованная модель
- Timestamp

Кэшируется на сервере. При повторном открытии страницы загружается из `/treasury/ai-cache`.

## Операции

### Update Balances

Кнопка **Update Balances** запускает фоновое обновление:

1. Отправляет POST-запрос на `/treasury/update?maxId={maxId}&concurrency={concurrency}`
2. Сервер запускает параллельное обновление балансов через DeBank API
3. Индикатор **Updating...** мигает
4. Статистика в header обновляется в реальном времени (polling каждые 1.5 сек через `/treasury/status`)
5. По завершении автоматически перезагружает данные

### Swap All

Кнопка **S** в строке аккаунта:

- Отправляет POST на `/treasury/swap-chains`
- Параметры: `id`, `chains` (выбранные в фильтре), `threshold`, `excludeStables`
- Выполняет swap всех токенов в нативный токен каждой сети

### Bridge To

Кнопка **B** в строке аккаунта:

- Отправляет POST на `/treasury/bridge-chains`
- Параметры: `id`, `chains`, `destination`, `threshold`
- Переводит токены из выбранных сетей в целевую сеть (Bridge to)

## Поддерживаемые блокчейны

Treasury поддерживает 40+ блокчейнов через DeBank API:

- **EVM**: Ethereum, BSC, Polygon, Arbitrum, Optimism, Base, Avalanche, Fantom
- **L2**: zkSync Era, Linea, Scroll, Manta, Mantle, Blast, Mode, Zora
- **Alt-L1**: Solana (через Monad), Aptos, Sui, Cosmos chains
- **Специализированные**: Ronin, Flare, Core, Kaia (Klaytn), Celo, xDai

Полный список с иконками хранится в константе `DEBANK_ICONS` (treasury.html:380-400).

## Edge cases

**Балансы не обновляются** — проверь что DeBank API доступен. Concurrency > 10 может вызвать rate limiting.

**Ячейка пустая, но токены есть** — сумма < $0.01. Hover покажет детали.

**AI Report не работает** — проверь что модель выбрана и `/ai-report/api/models` возвращает список. Если модель недоступна — выберите другую.

**Swap/Bridge не выполняется** — операции отправляются на сервер, но реальное выполнение зависит от backend-логики. Проверь логи сервера.

**Chain filter не применяется** — после изменения фильтра данные перерисовываются автоматически. Если не работает — нажми **Refresh**.

**Tooltip не показывается** — убедись что ячейка содержит данные (не пустая). Tooltip привязан к `data-tokens` атрибуту.
